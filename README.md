# SlotMachine-Game
C++로 개발한 슬롯머신 비주얼 게임

## 개발동기
랜덤함수를 이용한 프로젝트를 진행해 보고 싶어 개발.

## 기능소개
슬롯머신처럼 숫자가 내려가는 것처럼 보이게 함.
